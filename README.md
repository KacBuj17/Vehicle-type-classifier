# Vehicle-type-classifier
Vehicle type classification tool based on a lightweight simple convolutional neural network

How to run:
```
python3 -m venv .venv
.\.venv\Scripts\activate
pip install -r .\requirements.txt
```

To run on CUDA uninstall torch and torchvision installed from requirements.txt and install from:

```
https://pytorch.org/get-started/locally/
```
